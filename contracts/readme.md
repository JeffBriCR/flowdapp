# EOS Kanban Smart Contracts

EOS Kanban allows you to manage your workflow onchain. 

## Requirements

- [CMake](https://cmake.org/)
- [EOSIO](https://github.com/EOSIO/eos)
- [eosio.cdt](https://github.com/EOSIO/eosio.cdt)

### Code Standards

- 2 spaces
- use header files
- use namespaces

## Build 

`cmake . && make`
